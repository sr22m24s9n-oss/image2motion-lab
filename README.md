# image2motion-lab

1枚のイラスト・アニメ系画像から、AI補完込みで動画アニメーションを作るための
**無料ルートの発掘・検証・方法論化** を行うリポジトリ。

## 背景

- 知人から「この画像から動画を作りたい」という要望があった
- 元画像は現時点で手元にない（テスト用素材で先行して検証する）
- 有料サービス・有料APIは使わない（無料枠のみでルートを構築する）

## 前提・制約

| 項目 | 内容 |
| --- | --- |
| 素材 | イラスト / アニメ系画像（1枚絵） |
| 期待する動き | 未確定（軽いループ〜本格的な image-to-video まで調査対象） |
| 実行環境 | Intel HD Graphics 620 / i5-7200U — **ローカルGPU生成は不可** |
| 主戦場 | Webサービス無料枠 + Google Colab 無料枠 |
| コスト | 完全無料の範囲のみ |
| 最終利用者 | 未定（まず自分用に整備し、必要なら非技術者向けに清書） |

## 進め方

1. **調査** — 無料で使える image-to-video 手段の収集と現状確認（[docs/01-research](docs/01-research/)）
2. **対談・整理** — 要望の具体化、候補の絞り込み（[docs/02-requirements](docs/02-requirements/)）
3. **テスト実行** — テスト画像で各候補を実際に回し、記録する（[docs/03-experiments](docs/03-experiments/)）
4. **フィードバック修正・実装** — 有力ルートを確定し手順化（[docs/04-methodology](docs/04-methodology/)）
5. **振り返り** — 学び・落とし穴の記録（[docs/05-retrospective](docs/05-retrospective/)）
