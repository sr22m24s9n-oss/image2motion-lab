テスト01〜04の学びを織り込んだ完成形。日本語のままコピペで使う
（テスト03で日本語プロンプト成立を確認済み。エージェント基盤はGemini）。

## 事前設定（毎回の型）

- 生成モード: **材料から動画** — セレネのキャラ資産を材料に添付（固有性の固定。テスト03の学び）
- モデル: **Veo 3.1 Fast**（設定のスライダー型アイコンから。20cr/8秒）
- 縦横比: **9:16縦**（設定から。毎回確実に出るかは本テストの観測対象）
- エージェントが複数本の一括生成を提案してきたら**拒否して1本だけ**（テスト04の学び）

## ショット1: 夕暮れの雑踏（新規生成・歌なし）

```
夕暮れの旧市街の市場通り。広い引きのショットで街の全景から始まり、
カメラがゆっくり寄って、人混みの縁を歩く彼女を見つける。
彼女は20歳前後の落ち着いた大人びた女性として、フードを軽くかぶり、
石造りの建物が並ぶ通りの端を穏やかに歩いていく。
周囲の音: 市場の陽気なざわめき、行き交う足音、遠くの売り声。
画風は元イラストの手描き厚塗りダークファンタジー調のまま、
くすんだ大地色、夕暮れの柔らかい光。
```

設計メモ:
- 「追われる・怯える・人目を逃れる」の語は使わない（テスト01の安全フィルタ回避）
- 服装・顔の再説明はしない（材料が持つ。年齢と画風だけ毎回指定）
- 歌はまだ入れない（歌詞テストはショット3から）

## ショット2: 路地への進入（ショット1から**延長**で生成）

延長のプロンプトは「次の約7〜8秒で何が起こるか」だけを書く:

```
彼女は人波をゆっくり離れ、石造りの建物の間の狭く静かな路地へ
穏やかに入っていく。路地の奥で立ち止まり、壁に軽く背を預けて、
ほっと安堵の小さな息をつく。カメラは彼女を追い、ミディアムショットへ寄る。
音: 雑踏のざわめきが、路地に入るにつれて遠くくぐもっていき、静けさに変わる。
```

## 延長（Extend）の操作手順（Web調査 2026-07-15。実地で答え合わせする）

1. 生成したショット1のクリップを**シーンビルダーに追加**する
2. クリップを選択し、末尾の**「＋（追加）」→「拡張（Extend）」**を選ぶ
3. プロンプト欄に上のショット2の文を入れて生成
4. 続きを確認。必要ならさらに延長を重ねられる

仕組みと制約（調査値）:
- 延長は**直前クリップの最終約1秒（24フレーム）を起点**に生成される
  → キャラ・構図・光が物理的に引き継がれる。「連結は指示でなく機能で」の実体
- 1回の延長で伸びるのは**約7秒**。最大約20回・計約148秒まで
- **Veo 3.1以降は延長にも音声が付く**（旧版は音なしだった）
- 延長も毎回クレジット消費（Fast相当と想定。実消費は本テストで実測）
- Google AI Pro以上の限定機能（契約済みなので問題なし）
