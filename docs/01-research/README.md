# 01 調査 — 無料 image-to-video ルートの収集

## 調査観点

各候補について以下を埋める。

- 無料枠の内容（回数 / クレジット / 透かしの有無 / 商用可否）
- アカウント登録の要否・条件
- イラスト・アニメ系画像への適性
- 出力仕様（解像度・秒数・fps・形式）
- 生成の待ち時間・混雑状況
- 継続利用の現実性（無料枠が月次リセットか、使い切りか）
