# 2026-07-13 Google Flow — 領主より「使える」との情報。最有力候補に浮上

## 何者か

Google の AI 映像制作ツール。動画生成モデル Veo を搭載し、image-to-video に対応。
単発生成だけでなく**シーンビルダーでクリップの延長・連結ができる**のが特徴で、
「1分ショート = 複数クリップの結合」という本プロジェクトの構成と噛み合いが良い。

## 無料枠の調査値（2026年時点・Web調査。実地確認前）

| 項目 | 内容 |
| --- | --- |
| 無料クレジット | 月100（地域により最大180の報告あり）。月次リセット、使い切ると翌月まで生成不可 |
| Veo 2 Fast | 10クレジット/本 → **月10本程度**が無料の現実線 |
| Veo 3.1 | AI Pro（月2,900円）以上が必要 |
| クレジット追加購入 | 日本国内では未提供（2026-05時点）→ 月次リセット待ち運用 |
| 消費の変動 | モデル・処理内容で変わる。Quality系・4Kアップスケールは高消費 |

## 本プロジェクトへの当てはめ

- 1分ショート ≒ クリップ8〜12本 → **無料枠(月10本)でギリギリ1作品/月**。
  失敗リテイク分を考えると、Flow単独では足りない可能性が高い
- → **Flow(品質枠) + PixVerse/Kling/Vidu(日次無料枠で物量・リテイク) の併用構成**が現実的
- シーンビルダーで前クリップ末尾からの延長ができるなら、つなぎ目問題(キャラの一貫性)に強い

## 要確認（実地）

- [ ] 領主のアカウントで使えるのはどのプラン枠か（無料 / AI Pro）→ 使えるモデルと本数が決まる
- [ ] 無料枠出力の透かしの有無・解像度
- [ ] イラスト・アニメ系画像を入力したときの品質（Veoは実写評価が中心）
