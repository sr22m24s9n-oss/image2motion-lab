# 02 要望の整理 — 対談で詰めること

知人（依頼元）に確認する項目のリスト。回答が得られ次第このファイルに追記する。

## 確認事項

### 素材について
- [ ] 元画像の入手（可能なら現物、無理なら「似た系統のサンプル」）
- [ ] 画像の権利関係（自作か、二次利用か → 使えるサービスの規約に影響）
- [ ] 解像度・縦横比

### 動きについて
- [ ] どんな動きを想像しているか（まばたき程度 / カメラワーク / キャラが大きく動く）
- [ ] 尺（何秒くらいか、ループでよいか）
- [ ] 音は必要か（BGM・リップシンク）

### 完成品について
- [ ] 用途（SNS投稿 / 個人鑑賞 / その他 → 透かし許容度・商用可否に影響）
- [ ] 求める品質ライン（多少の崩れは許容できるか）
- [ ] 納期感

## テスト用素材

元画像の入手前は、権利的に問題のない自作 or フリー素材のイラストで検証する。
素材は `../../assets/` に置き、出所を必ず記録する。
