# 02 要望の整理 — 対談で詰めること

知人（依頼元）に確認する項目のリスト。回答が得られ次第このファイルに追記する。

## 確定済み（2026-07-13）

- **用途: YouTube用の1分ショート**
- テスト素材: 「セレネ」の画像を使う予定（権利・入手経路は下記で確認中）

### 「1分ショート」が構成に与える影響

- AI image-to-video の1回の生成は **5〜10秒/本** が相場。1分には届かない。
  → 方法論は「**複数クリップ生成 + 編集で結合**」が前提になる（クリップ6〜12本 + カット編集・BGM）
- 1日の無料クレジット ≒ 1〜数本 のサービスが多いため、**1本の完成に数日かかる**運用を織り込む
- YouTube公開＝人目に触れる用途なので、**透かしは極力避けたい**
  → 透かしなしで出せる無料枠（Vidu オフピーク / Colabルート）の優先度が上がる
- 収益化する場合は各サービスの無料枠の商用利用可否が効いてくる（要確認）
- 編集結合ツールも無料で必要: Canva（生成と編集を一括）/ CapCut などが候補。
  ローカルPCが非力なため、ブラウザ完結型を優先する

## 確認事項

### 素材について
- [ ] 元画像の入手（可能なら現物、無理なら「似た系統のサンプル」）
- [ ] 画像の権利関係（自作か、二次利用か → 使えるサービスの規約に影響）
- [ ] 解像度・縦横比

### 動きについて
- [ ] どんな動きを想像しているか（まばたき程度 / カメラワーク / キャラが大きく動く）
- [ ] 尺（何秒くらいか、ループでよいか）
- [ ] 音は必要か（BGM・リップシンク）

### 完成品について
- [x] 用途 → **YouTube 1分ショート**
- [ ] 収益化の予定はあるか（無料枠の商用可否に影響）
- [ ] 求める品質ライン（多少の崩れは許容できるか）
- [ ] 納期感

### テスト素材「セレネ」について
- [ ] セレネとは何か（自作キャラ / 依頼絵 / 既存作品のキャラ）→ 権利確認
- [ ] 画像ファイルの受け渡し
- [ ] 公開リポジトリに置いてよいか（NGなら手元管理のみで、記録はテキストだけにする）
